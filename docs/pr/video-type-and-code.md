## 一、生动形象的比喻

H.264 = 压缩方式（怎么“折叠衣服”）；MP4 / MOV / QuickTime = 容器（用什么“箱子装”）

## 二、H.264（编码方式——压缩算法）

这就相当于是 ==折衣服的方法==。

想象你要搬家，要把衣服塞进箱子；而 H.264 就是“怎么折衣服更省空间”的技巧。

特点：1、把视频“压缩”得更小；2、画质还不错（性价比高）；3、是目前最常见的视频编码方式之一。

:::tip 它不是文件格式，是“编码方式”。
:::

## 三、MP4（常用“纸箱”）

就像普通纸箱，谁都能用；几乎所有设备都支持。

特点：1、兼容性最强；2、通常里面装的是 H.264 视频。

::: info 常见组合：MP4 + H.264（黄金搭档）。
:::

## 四、QuickTime：苹果的“搬家体系”

这是 QuickTime 的一整套多媒体系统；类似“苹果专用搬家公司”。

它包括：1、播放器；2、文件格式（MOV）；3、编解码支持。

:::tip 它是一个体系，不只是格式。
:::

## 五、MOV：苹果专用箱子

MOV 是 QuickTime 体系里的“箱子”，类似“苹果定制纸箱”。

特点：1、更适合专业视频（剪辑、后期）；2、文件通常更大；3、兼容性不如 MP4。

## 六、结构关系

视频文件 = 容器（MP4 / MOV） + 编码（H.264）

举例：

:::table align="center"
|    文件     |      实际含义       |
|:---------:|:---------------:|
| video.mp4 |    用 MP4 箱子装    |
| video.mov |    用 MOV 箱子装    |
|    内容     | 很可能都是 H.264 压缩的 |
:::
