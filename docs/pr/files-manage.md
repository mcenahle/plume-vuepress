::: note 请区分“文件管理”和“项目管理”
本页面讨论的是 **文件管理**，即：如何在资源管理器中整理 Premiere Pro（PR）项目文件，使其结构清晰、便于查找与打开。

如果你想了解的是 **项目管理**（例如：完整打包项目、归档或发送给他人），请参考：[如何进行项目管理](project-manage)。
:::

PR 文件管理要以可便于分类、打开为根本目的。使用本规范性纲要，即可方便的管理 PR 文件：

::: file-tree icon="colored"

- D:\pr
  - 课程一  # 一个**项目**目录
    - 素材
      - 视频
        - 1.mp4
        - 2.webm
        - 3.mov
      - 音频
        - 1.mp3
        - 2.mp3
      - 其它素材       # 非视频、音频素材
        - picture.jpg
        - picture2.png
        - word.docx
        - powerpoint.pptx
    - 导出
      - export.mp4
    - 课程一.prproj
  - 课程二  # **另一个**项目
    - 素材
      - 视频
        - 1.mp4
        - 2.webm
        - 3.mov
      - 音频
        - 1.mp3
        - 2.mp3
      - 其它素材       # 非视频、音频素材
        - picture.jpg
        - picture2.png
        - word.docx
        - powerpoint.pptx
    - 导出
      - export.mp4
    - 课程二.prproj
:::
