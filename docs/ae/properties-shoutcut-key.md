:::table align="center"
|    按键     |      控制项       |
|:---------:|:---------------:|
| P |    位置（Position）   |
| S |    缩放（Scale）      |
| R |    旋转（Rotation）   |
| T |    不透明度（Transparent）   |
| A |    锚点（Anchor Point） |
:::

1. 已经单独显示属性后，按住 `Shift` + 上述快捷键，可以叠加显示多个属性。例如：`P` + `Shift` + `S`，可同时显示位置 + 缩放；
2. 连续按两次 `U`，可显示所有被修改过的属性。

:::tip P位 S缩 R旋 T透 A锚
:::
